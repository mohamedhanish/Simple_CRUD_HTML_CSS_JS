# SIMPLE CRUD HTML CSS JS

# ENGLISH BELOW ⬇️

هذا المشروع بسيط ومصمم للتدريب على تعلم أساسيات JavaScript وعمليات CRUD (إنشاء، قراءة، تعديل، حذف) بالإضافة إلى البحث بين البيانات.  
المشروع يستخدم HTML وCSS وJavaScript فقط بدون أي مكتبات خارجية، ومناسب لتعلم أساسيات تطوير الويب.

-----------------------------------------------------------

This project is simple and designed for practicing basic JavaScript and CRUD operations (Create, Read, Update, Delete), including searching data.  
The project uses only HTML, CSS, and JavaScript without any external libraries, and is suitable for learning basic web development.
